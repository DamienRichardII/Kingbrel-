Guide rapide EmailJS

  1. Crée un compte sur emailjs.com
  2. Ajoute un Service email (Gmail, Outlook…) → note le Service ID
  3. Crée un Template avec les variables : {{to_email}}, {{client_name}}, {{service_name}}, {{date}}, {{start}}, {{end}}, {{client_phone}}, {{client_email}}, {{note}}
  4. Copie ton Public Key depuis Account → API Keys
  5. Colle les 3 valeurs ci-dessus et enregistre
